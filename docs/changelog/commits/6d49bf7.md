## Description

This commit implements a comprehensive fleet management system for NEOPRO Raspberry Pi devices, allowing centralized control from headquarters.
## Phase 1: Central Server (100%)
- Node.js + Express + TypeScript backend
- PostgreSQL database with complete schema
- REST API for auth, sites, and groups management
- WebSocket (Socket.IO) for real-time communication
- JWT authentication with RBAC (admin, operator, viewer)
- Automatic alerts (temperature, disk, memory)
- Deployment config for Render.com (~$14.50/month)
## Phase 2: Sync Agent (100%)
- Node.js agent for Raspberry Pi devices
- WebSocket client connecting to central server
- 8 commands: deploy_video, delete_video, update_software, update_config, reboot, restart_service, get_logs, get_system_info
- System metrics collection (CPU, RAM, temperature, disk, uptime)
- Automatic heartbeat every 30 seconds
- Automatic rollback for failed updates
- systemd service configuration
- Installation and registration scripts
## Phase 3: Angular Dashboard (95%)
### Architecture (100%)
- Angular 17 standalone components
- Complete services: API, Auth, Socket, Sites, Groups
- Auth guards and HTTP interceptors
- TypeScript models for all entities
- Lazy loading routing
- Environment configs (dev + prod)
