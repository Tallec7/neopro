## Description

Cloud providers like Render, Supabase, and Neon use certificates that may not
be in the system CA store. Changed to use rejectUnauthorized: false by default
when DATABASE_SSL_CA is not set, fixing the "self-signed certificate in
certificate chain" error on Render deployment.
